Selected Academic Projects

IoT Project (2015)
We created a platform that controls a cluster of programmable sensors. It elects periodicaly one sensor to be the data aggregator, in order to minimize the cluster's energy consumption. The sensors listened to commands through Sockets and were programmed in C++ using RIOT OS.

Aladeen Project (2015)
We created a platform similar to Shazam that given a video segment it finds its origin from a video database. We offered an application to load video segments and a Server with a video database. It was programmed in C++ using Qt, pHash and Sockets to connect the Client and the Server

BitTorrent Client (2014)
We created a full featured BitTorrent client and tracker that had the ability to connect to many other clients and transfer various files simultaneously, using JAVA threads, Sockets and Swing.

3D Graphics Engine (2012-Present)
It uses deferred shading techniques to render objects and shadow maps for the shadowing. It also supports Dynamic Lights, BillBoards, Particle System, 2D Gui, Enviroment Mapping, Normal Mapping, Post-Processing Effects, View-Frustum Culling and an Obj / Mtl loader.

We created the classic Super Mario game for the Android Platform. It featured 3 Mario transformations, enemies and a number of levels. We also used profiling and java memory analyzers to counter a few bottlenecks in our program

AirShip Fighters (2011)
I created a 3D game using C++ and OpenGL. The player controls an airship and should get the treasure in order to win. There are randomly spawned enemies who are trying to take him down. The player can shoot down the enemies using a mounted cannon on the airship. The game also featured a weather and particle system.
